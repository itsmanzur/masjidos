FAQ

Questions committees ask

Short answers. Full detail lives in the in-plugin Docs.

Is MasjidOS free?

Yes. The core plugin is free on WordPress.org. Install, configure prayer times, and publish shortcodes without a paid plan.

Do prayer times need the internet?

No for the default path. Times calculate on your WordPress site from coordinates and method. An optional online API is available if you prefer it.

Can we run a lobby TV?

Yes. Open the dedicated display URL (default /masjidos-display/) on a Chromebox or smart TV browser in fullscreen.

Which languages are supported?

Admin UI and public widgets support English, Bangla, and Arabic. You can set admin and public languages separately.

Do we need a developer?

Usually not. Welcome, Features, and Docs walk committees through setup. Shortcodes paste into any page builder or classic editor.
