# Self-hosted MasjidOS fonts

Shared by Admin UI, public widgets, and TV Display (`tv-fonts.css`).

- Outfit — SIL Open Font License 1.1 (Latin UI)
- Cairo — SIL Open Font License 1.1 (Arabic UI)
- Noto Sans Bengali — SIL Open Font License 1.1 (Bangla UI)

No requests are made to fonts.googleapis.com or fonts.gstatic.com at runtime.
